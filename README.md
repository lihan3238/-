- 钓鱼: 游历最多好像10个，炼灵阵好像是25个

## 家族战经验

1. 第一个城让非活跃玩家慢慢打，活跃玩家攒票不要打等通知统一指挥；
2. 每次开打前提前3小时统计票数，预定时间统一开打；
3. 打的时候，预留随时盯着的3人小队一人20票等城市到斩杀线秒掉；
4. 我方高战留票不要打，打高战或者驻守；
5. 低战磨对面高战35次疲劳；
6. 开战前，尽量联络非安天家族合纵。
